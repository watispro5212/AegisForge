about aegisforge

A high-performance Discord bot engineered from the ground up in Rust: purpose-built for speed, reliability, and deep server control without unnecessary bloat.

AegisForge is built for communities that need dependable tools. Moderation, economy, leveling, and utility workflows are designed to stay fast, readable, and consistent as a server grows.

Rust gives the bot predictable performance and memory safety. Tokio handles concurrent work, Serenity and Poise power Discord interactions, SQLx manages PostgreSQL persistence, and DashMap keeps hot guild configuration data quick to access.

Whether you run a small friend server or a busy public community, AegisForge keeps the basics close at hand: clear commands, persistent settings, live stats, and practical automation.

What aegisforge Does

🛡️

Moderation

Full suite of moderation commands — ban, kick, mute, timeout, warn, purge, slowmode, lock — all with structured embed responses and a persistent case numbering system for accountability.

💰

Economy

A complete server economy with wallets, banking, daily rewards, gambling (slots), rob mechanics, and rich leaderboards — all persisted per guild in PostgreSQL.

📈

Leveling

Automatic XP tracking per message, configurable level-up announcements, and role rewards at specific milestones — keeping your most active members engaged and recognized.

⚙️

Configuration

Every guild gets fully customizable settings — welcome/goodbye channels and messages, auto-role on join, moderation log channels, and per-server command prefix.

⚡

Utility

Dozens of quality-of-life commands: server info, user lookup, avatar fetching, QR code generator, math evaluator, world clock, reaction polls, Discord timestamp formatter, and more.

🎉

Fun

Keep your community entertained with trivia, coinflip, dice, 8-ball, ship, rate, mock text, animal pictures, and 20+ social interaction commands.

Technology Stack

🦀 Rust Systems language — memory-safe, zero GC, blazing fast
⚡ Tokio Async runtime for non-blocking concurrent task handling
🔷 Serenity Low-level Discord API client with full gateway support
🎯 Poise Slash command and prefix command framework on top of Serenity
🐘 SQLx Compile-time verified async PostgreSQL queries
🗄️ Neon Serverless PostgreSQL with connection pooling via PgBouncer
🔑 DashMap Lock-free concurrent hashmap for in-memory guild config cache
🌐 Axum High-performance web server powering the /api/stats endpoint
