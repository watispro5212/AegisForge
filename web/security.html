Layer 1

Sentinel Anti-Raid

Detect coordinated join spikes in real time. When the join rate exceeds your configured threshold, AegisForge can auto-kick the users who triggered the window and post an alert to your mod log.

Fast response

Join windows are processed in memory so the bot can react without waiting on a database round-trip.

Configurable thresholds

Set the joins and time window that fit your community's normal traffic.

Logged actions

Every raid response can be routed into your mod log with counts and target details.

/sentinel enable /sentinel threshold joins:5 window:10 /sentinel status

Layer 2

AutoMod Suite

Enable the modules your server needs: spam filtering, invite blocking, caps detection, mass mention cleanup, and blacklist phrases. Each module can be tuned without redeploying the bot.

Anti-Spam/automod spam on

Deletes repeated messages sent too quickly.

Anti-Invite/automod invites on

Blocks Discord invite links across channels.

Anti-Caps/automod caps on

Removes messages dominated by uppercase text.

Anti-Mention Spam/automod mentions on

Deletes messages with too many user or role mentions.

/automod enable /automod blacklist-add phrase:text /automod status

Layer 3

Shadow Ban System

Silently restrict a member without sending a DM or public callout. The mute role is applied in the background and the action is logged for staff review.

Silent restriction

The target is not notified, keeping staff response calm and controlled.

Moderator-visible response

Confirmation can stay private to the executing moderator.

Persistent record

The action is stored and tied into moderation history.

/muterole role:@Muted /shadowban user:@user reason:text /unshadowban user:@user

Layer 4

Tactical Commands

High-impact operations for active incidents: lock the server, restore normal access, pull a user's moderation history, or kick with message cleanup.

/tactical intercept

Locks every text channel during active raids or emergencies.

/tactical restore

Unlocks channels and returns the server to normal.

/tactical report @user

Shows moderation history, action counts, and recent cases.

/tactical breach @user

Kicks the user and bulk-deletes recent messages from the channel.

Observability

Full event logging

Every significant server event is captured, timestamped, and routed to your dedicated log channels.

Mod Log

Moderation actions, AutoMod hits, and Sentinel triggers logged with moderator, target, reason, and case number.

/logs channel:#mod-log

Message Log

Message edits and deletes show useful context for review without cluttering public channels.

/msglogs channel:#msg-log

Member Log

Member joins and leaves can include account age, user ID, and server context.

/memberlogs channel:#member-log
Quick Setup

Security in 5 commands

Get the core protection stack running in a few minutes.

01

Set your log channels

/logs channel:#mod-log/msglogs channel:#msg-log/memberlogs channel:#member-log
02

Enable Sentinel

/sentinel enable/sentinel threshold joins:5 window:10
03

Turn on AutoMod

/automod enable/automod spam on/automod invites on
04

Configure the muted role

/muterole role:@Muted

Create a role with Send Messages denied, then assign it here.

05

Done: you're protected
