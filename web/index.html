Knowledge Base

Frequently Asked Questions

Why Rust?

Rust gives the bot predictable memory usage, strong compile-time checks, and excellent async performance.

Is it free?

Yes. AegisForge is free to add and open source.

How do I setup logs?

Use /logs channel:#channel to designate where all moderation actions should be archived.

How does Sentinel work?

Run /sentinel enable and optionally /sentinel threshold joins:5 window:10. When the join rate exceeds the threshold, joiners are auto-kicked and a raid alert fires to your mod log channel.

What is a shadow ban?

Use /muterole @role once to set it up, then /shadowban @user. The mute role is applied silently — no DM or public message. Only your mod log sees the action.
