Command Reference

/mod banPermanently ban a member with optional message purge
/mod unbanUnban a user by ID
/mod kickRemove a member from the server
/mod timeoutMute a member for a specified duration
/mod warnIssue a formal warning logged to the database
/mod purgeBulk delete up to 100 messages in a channel
/util pingCheck API round-trip latency
/util serverDisplay detailed server information
/util userLook up information on any user
/util avatarRetrieve a user's full-resolution avatar
/util uptimeView bot uptime and version details
/util timestampGenerate Discord timestamp formats from UNIX time
/role addAssign a role to a member
/role removeRemove a role from a member
/role listList all roles in the server by hierarchy
/config logsSet the moderation log channel
/config welcomeSet the welcome message channel and template
/config autoroleSet the role auto-assigned to new members
/remind createSet a reminder delivered to you after N minutes
